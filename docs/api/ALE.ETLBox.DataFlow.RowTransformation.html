Class RowTransformation

Transforms the data row-by-row with the help of the transformation function. The non generic RowTransformation accepts a string array as input and returns a string array as output. If you need other data types, use the generic RowTransformation instead.

Examples
//Non generic RowTransformation works with string[] as input and output
//use RowTransformation<TInput,TOutput> for generic usage!
RowTransformation trans = new RowTransformation(
    csvdata => {
      return new string[] { csvdata[0],  int.Parse(csvdata[1]) };
});    
trans.LinkTo(dest);
